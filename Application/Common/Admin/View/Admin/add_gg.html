添加广告图 图片只支持jpg png jpeg格式
广告图片格式说明如下：

一:主页：
  1. banner广告一(首页轮播图下 直播间上 )：1100*140
  2. 底部广告展示图（首页最底下的广告图）：1100*140
  3. 直播部分左一展示广告（首页直播间下的左侧）：260*500
  4. 首页左右2侧广告图 210*510 左侧填写在大图里 右侧填写在小图里
二:直播间首页:
  1. PC大屏顶部主体广告展示图：1900*499
  2. 移动小屏顶部主体广告展示图：1080*460
三、直播间部分
  1. 中部广告展示图：1060*200
四、精选视频（点播）首页
  1. PC大屏顶部主体广告展示图：1900*499
  2. 移动小屏顶部主体广告展示图：1080*460
五、精选视频（点播）播放部分
  1. 中部广告展示图：1060*200
注意：
第二大项和第四大项根据分类需要传两张不同大小尺寸的图片，其余项根据分类标注的大小传相同尺寸。
即 下面下拉列表写着需要传2张图的 需要准备2种尺寸，如果没说明则传相同尺寸。
